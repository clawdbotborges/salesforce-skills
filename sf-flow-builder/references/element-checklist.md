# Element Quick Checklist

## Assignment
- [ ] Label and API Name
- [ ] At least one variable-operator-value row
- [ ] Variable exists (create first if needed)
- [ ] Value type matches variable type

## Decision
- [ ] Label and API Name
- [ ] At least one custom outcome with conditions
- [ ] Default outcome labeled descriptively
- [ ] Condition logic set (All/Any/Custom)
- [ ] Values reference existing variables/resources

## Get Records
- [ ] Label and API Name
- [ ] Object selected
- [ ] Filter conditions set (at least one)
- [ ] Sort configured if order matters
- [ ] How many: First Record / All Records
- [ ] Storage: "Choose fields" for agent flows
- [ ] Specific fields selected (don't store all)
- [ ] Assigned to correct variable/collection
- [ ] Fault path with error handling

## Create Records
- [ ] Label and API Name
- [ ] Object selected
- [ ] Required fields populated
- [ ] Field values reference correct variables
- [ ] Fault path with error handling

## Update Records
- [ ] Label and API Name
- [ ] Find method selected (by variable or by conditions)
- [ ] If by conditions: object + filter set
- [ ] Field values to update specified
- [ ] Fault path with error handling

## Delete Records
- [ ] Label and API Name
- [ ] Find method selected
- [ ] Conditions correctly identify target records
- [ ] Fault path with error handling

## Screen (Screen Flows only)
- [ ] Label and API Name
- [ ] Components added and configured
- [ ] Required fields marked required
- [ ] Navigation buttons configured (Next, Previous, Finish)

## Loop
- [ ] Label and API Name
- [ ] Collection variable selected
- [ ] Loop variable (current item) configured
- [ ] Direction: First to Last (usually)
- [ ] Elements inside loop added
- [ ] After-loop path has elements if needed

## Action / Subflow
- [ ] Label and API Name
- [ ] Action/subflow selected
- [ ] Input values mapped
- [ ] Output values captured to variables
